# Requerimientos del Software de Cajero

## 1. Gestion de turno y caja

- abrir caja con fondo inicial
- ver historial en vivo del turno actual
- calcular cuadratura por efectivo, tarjeta y transferencia
- cerrar caja para revision administrativa

## 2. Proceso de venta

- escaneo rapido con pistola de codigo de barras
- busqueda manual por nombre
- sumar, restar y quitar productos del carrito
- aplicar precio mayorista y pallet automaticamente
- soportar productos pesables con fracciones

## 3. Pagos y documentos

- cobrar con efectivo, tarjeta y transferencia
- emitir boleta electronica
- emitir factura electronica
- emitir vale interno
- crear cliente express para facturacion

## 4. Despachos

- asignar carga a chofer o transportista
- escanear productos del camion
- emitir guia o vale para rendicion

## 5. Hardware adicional

- impresion automatica en termica 80mm
- segunda pantalla para cliente
- mostrar publicidad cuando la caja esta inactiva

## 6. Requisito operativo clave

El sistema debe ser a prueba de errores comunes, rapido y usable casi al 100% con teclado o escaner.
